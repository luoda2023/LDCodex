服务重启

通过批处理文件或 manage.mjs 管理 LUODA 中转代理服务的启停操作。

方式一：批处理文件启停 推荐

在项目根目录下依次执行以下批处理文件：

1. 停止服务.bat —— 停止所有正在运行的代理服务
2. 启动后台服务.bat —— 重新启动代理和管理面板服务

• 两个文件需在 同一 cmd 窗口 中依次执行

• 停止服务后请等待进程完全退出（约 2-3 秒），再执行启动脚本

方式二：manage.mjs 独立管理 灵活

使用 manage.mjs 脚本可以独立重启管理后台服务，无需影响代理转发：

# 重启管理后台（不影响代理转发）
node manage.mjs restart

• 适用于仅需刷新管理后台页面功能时的快速重启

• 不会中断正在进行的 API 请求转发

完整重启流程
■
停止服务
执行 停止服务.bat
⏳
等待退出
确认进程已完全终止
▶
启动服务
执行 启动后台服务.bat
