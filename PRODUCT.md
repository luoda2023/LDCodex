# LDCodex2

## Register
product

## One-liner
多平台 AI 编程助手增强管理工具，管理 Codex 的运行实例、配置和状态监控。

## Target Users
- 使用 Codex CLI 的开发者
- 需要在多个 Codex 实例间切换管理的高级用户
- 关注服务状态、Token 统计和配置管理的 AI 辅助开发人员

## Product Purpose
LDCodex2 是一个 Tauri 桌面应用，提供对 Codex CLI 底层进程的管理能力。
让用户能够启动/停止服务、查看运行状态、管理 Provider 预设、查看 Token 统计，
以及执行配置管理和生命周期控制。

## Brand Personality
- 专业（Professional）：面向开发者的工具，界面干净利落
- 沉稳（Grounded）：深色主题，低调不张扬
- 高效（Efficient）：信息密度高，操作路径短
- 可靠（Reliable）：状态指示清晰，反馈明确

## Anti-References
- 不要过度动画：UI 应该响应快速，不做无意义的动效
- 不要圆润可爱：这是开发工具，不是消费级应用
- 不要用渐变做主色调：保持单色系的品牌色使用
- 不要大面积的纯白背景：以深色模式为主

## Strategic Design Principles
1. 信息密度优先 — 开发者习惯一眼扫到大量信息，不要过度留白
2. 状态即交互 — 每个状态指示器都应该是可操作的入口
3. 深色模式为主 — Light 模式兼容但不作为主要设计目标
4. 本地化感知 — 中英文双语界面
5. 保持工具感 — 不要试图做漂亮，要做得好用
